## Running it locally

First, add your Spotify Client ID and Client Secret to `.env.local`

Second, install the dependencies and run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
